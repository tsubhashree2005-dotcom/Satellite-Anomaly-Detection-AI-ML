# Satellite Anomaly Detection using AI/ML

## Overview

This project presents an AI/ML-based anomaly detection system for satellite and spacecraft checkout data. It analyzes spacecraft telemetry and command data to identify abnormal system behavior, assisting in intelligent health monitoring and early fault detection.

The project combines data preprocessing, machine learning, real-time telemetry analysis, visualization, and reporting into a complete anomaly detection pipeline.

---

## Features

- Telemetry data preprocessing
- Command and telemetry analysis
- AI/ML-based anomaly detection
- Real-time spacecraft monitoring simulation
- Model evaluation and comparison
- Interactive state diagram visualization
- Automated JSON and CSV report generation
- Performance visualization
