# Satellite Anomaly Detection using AI/ML

## Overview
An AI/ML system for detecting anomalies in spacecraft telemetry and checkout data to improve satellite health monitoring.

## Features
- Telemetry preprocessing
- Feature engineering
- Machine learning-based anomaly detection
- Performance evaluation
- Visualization of anomalies

## Technologies
- Python
- NumPy
- Pandas
- Scikit-learn
- PyTorch/TensorFlow (whichever you used)
- Matplotlib

## Dataset
Spacecraft checkout and telemetry data.

## Results
- Achieved high anomaly detection accuracy.
- Reduced false alarms through intelligent monitoring.
